projects
========

An API that provides a description of my personal projects.

Usage
-----

.. code-block:: bash

    $ curl projects.jakegillespie.me
    "Hello, world!"

Development
-----------

*Note that Docker is a prerequisite*

Start the application in development mode with

.. code-block:: bash

    $ ./bin/start
    $ curl localhost:8000
    "Hello, world!"
